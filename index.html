「题解」洛谷5278算术天才⑨与等差数列

Posted on 2020-03-04 | In 数据结构

非正解过题~

为方便起见，设 $b=\min\limits_{i\in[l,r]}\{a_i\}$，$b=\max\limits_{i\in[l,r]}\{a_i\}$。

前置基础知识

$\min\&\max$

具体而言，是用线段树维护区间 $\min$ 和区间 $\max$，只要 $b-a=k(r-l)$ 即认为正确。

至少能过样例……

真是佩服这数据，居然能过 #4, #6, #7, #8, #10，拿到 50 分的高分……

评测记录

Hack：

$\min\&\max\&\sum a_i$

多维护区间和，除了上述条件以外，还需 $\sum\limits_{i=l}^ra_i=\dfrac{(r-l+1)(a+b)}{2}$ 才认为正确。

在代码里，我使用 int 自然溢出来完成。（取模效率低）

你可能会问，怎么除以 $2$？很简单，直接看 sum * 2 是不是等于 (r - l + 1) * (a + b) 就行了。

居然就过了……

评测记录

Hack：

$\min\&\max\&\sum a_i\&\sum a_i^2$

多维护区间平方和，除了上述条件以外，还需 $\sum\limits_{i=l}^ra_i^2=\sum\limits_{i=0}^{r-l}(a+ki)^2$ 才认为正确。

我们依然可以用 int 自然溢出那个 trick。

评测记录

不会 hack……

不过，如果只维护 $\min\&\max\&\sum a_i^2$ 的话，还是会 hack 的（因为 $7^2+24^2=625=15^2+20^2$）：

实际上，维护 $\min\&\max\&\sum a_i\&\sum a_i^2$ 应该还是能被 hack 的，如果有谁找到了 hack 数据可以发在评论区里让大家开开眼界。

$\min\&\max\&\sum a_i\&\sum a_i^2\&\sum a_i^3$

多维护区间立方和，除上述条件以外，还需 $\sum\limits_{i=l}^ra_i^3=\sum\limits_{i=0}^{r-l}(a+ki)^3$ 才认为正确。

（实际上，如果维护的数据足够多，就能获得成就：在数据范围内都无法找到能 hack 你的数据，这下子这个非正解就秒变正解了，不过常数嘛……）

提交记录

「2020专题训练笔记」0-目录

Posted on 2020-02-19 | In 2020专题训练

前言

初一的时候，我“学习”了很多 OI 知识，不乏有数据结构、图论、数学的内容。不过，在学习的时候我比较急于求成，基本是都只知道个大概，数据结构的思想都大致明白，图论的内容一知半解。一到做题，就只会做模板题，甚至模板题都不能自己实现，得靠面向代码编程的方法来混个 AC。

这种相当错误的学习方式，在 CSP-J/S2019 中就立刻显出了弱点：代码能力不行，知识点不会灵活运用，就算想到了用什么知识点也很难写出来。当然，成绩是非常凄惨的，GD 省选资格都没有。

为了不重蹈旧辙，我决定在写这个学期的专题训练的时候，学习时归纳知识点，顺便简单地写写题解，做题想不出来才看题解，希望这种方式能够帮到自己。
